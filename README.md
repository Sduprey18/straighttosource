# Right to Source

A tiny, dependency-free web app that turns a Jobright or Simplify listing into the original employer or ATS posting.

```bash
npm start
```

Open `http://localhost:3000`.

## Deploy to Netlify

Connect this repository in Netlify and deploy. `netlify.toml` configures `public` as the published site and routes `/api/resolve` to the bundled resolver function automatically.
